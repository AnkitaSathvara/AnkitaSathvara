The Gazetteer

"The Gazetteer" is an interactive world map project powered by the LeafletJS library. Its primary purpose is to provide users with a dynamic map where they can overlay various points of interest. Additionally, it offers convenient popup modals that can be triggered to display country-specific information. This functionality is achieved through multiple API calls, enabling users to access detailed data for specific countries.

Company Directory

The "Company Directory" project is designed to simulate a company database environment. This project leverages AJAX calls to interact with PHP scripts, facilitating seamless communication with a SQL database. Through this system, it becomes possible to perform a wide range of operations, including Create, Read, Update, and Delete (CRUD), while ensuring the database's integrity and consistency.
